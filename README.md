# Healthcare Patient Risk Analysis

## Project Overview

This project analyzes synthetic electronic health record (EHR) data to identify factors associated with patient outcomes, healthcare costs, and readmission risk.

## Dataset

- patients.csv
- patient_outcome.csv

## Methods

### Data Cleaning
- Missing value analysis
- Outlier detection

### Feature Engineering
- Risk indicators
- Derived healthcare metrics

### A/B Testing
- Group comparison analysis
- Statistical significance testing

### Bootstrap Simulation
- Confidence intervals
- Uncertainty estimation

### PCA
- Dimensionality reduction
- Identification of latent factors

### Clustering
- Patient segmentation
- Risk group discovery

## Key Findings

- Identified high-risk patient segments.
- Evaluated readmission drivers.
- Quantified uncertainty using bootstrap methods.
- Reduced feature complexity using PCA.

## Business Impact

Healthcare organizations can use these insights to:

- Reduce readmission rates
- Improve resource allocation
- Identify high-risk patient populations
- Support data-driven intervention strategies
